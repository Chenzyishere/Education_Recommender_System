# 第四章 实验设计与分析

## 4.1 数据集预处理

### 4.1.1 数据来源与字段

本研究使用 ASSISTments 风格学习行为数据，核心原始文件为 `data/skill_builder_data.csv`。  
经预处理后生成 `data/assist9_cleaned.csv`，主实验使用字段为：

1. `user_id`：学生编号  
2. `skill_id`：知识点编号（重映射后连续）  
3. `correct`：作答结果（0/1）

时间特征 `time_gap` 在当前主实验中不启用（`USE_TIME_GAP=False`），仅作为可选消融方向保留。

### 4.1.2 数据清洗流程

根据 `preprocess/clean_data.py`，清洗流程如下：

1. 读取原始数据并统一列名格式（去除空格）。  
2. 保留核心列 `user_id, skill_id, correct, order_id`。  
3. 删除 `skill_id` 缺失样本。  
4. 去重并按 `user_id + order_id` 排序，保证行为时序。  
5. 将原始知识点 ID 映射为连续 ID（从 1 开始）。  
6. 导出映射文件 `data/skill_map.csv` 与清洗文件 `data/assist9_cleaned.csv`。  

该流程保证了序列模型输入的一致性，并为后续 KG 邻接矩阵构建提供稳定索引。

### 4.1.3 训练/验证/测试划分与训练策略

本研究采用用户级划分，避免同一学生行为同时出现在训练与测试中造成信息泄露。  
在 `train_and_eval.py` 中，整体比例为：

1. 训练集约 72% 用户  
2. 验证集约 8% 用户  
3. 测试集约 20% 用户  

训练策略包括：

1. 统一最大序列长度 `MAX_SEQ=100`。  
2. 验证集早停（`EARLY_STOPPING_PATIENCE=5`）。  
3. 以验证集最优轮次回载模型，再在测试集报告最终结果。  
4. KG-SAKT 采用动态逻辑损失权重调度（`LogicLambda` 退火）。

## 4.2 知识图谱构建

### 4.2.1 图谱构建方法

本研究通过 `preprocess/kg_builder.py` 生成知识图谱邻接表 `data/kg_adj_list.json`。  
当前构建策略为实验型轻量规则：当知识点 ID 大于 5 时，默认其前置知识点为 `skill_id - 5`。  
即：

$$
pre(s) = s-5, \quad s > 5
$$

并据此构建有向先修关系图。

### 4.2.2 图谱表示与模型接入

在训练阶段，邻接表被转化为稠密邻接矩阵 `kg_matrix`，其含义为：

1. 行：目标知识点  
2. 列：前置知识点  
3. 值：是否存在先修依赖（0/1）

KG-SAKT 使用该矩阵进行：

1. 一阶与二阶图扩散表示构建；  
2. 逻辑一致性损失计算；  
3. 路径合规性相关指标评估。  

### 4.2.3 构建方式的局限性

需要指出，当前 KG 是规则生成图，而非专家标注或课程大纲抽取图。  
该设定有利于算法验证，但会限制模型上限与外部有效性。  
因此，本章实验结论主要证明“引入结构先修约束具备可行性”，而非宣称图谱本身已达到最优教育知识结构。

## 4.3 实验结果分析

### 4.3.1 指标体系

为同时衡量预测能力与推荐路径逻辑性，本文采用以下指标：

1. `AUC`：下一题正确性预测能力（越高越好）  
2. `RMSE`：预测误差（越低越好）  
3. `Path`：路径完全合规率（越高越好）  
4. `PVR`：前置违规率（越低越好）  
5. `APC`：前置覆盖率（越高越好）  
6. `VS`：违规严重度（越低越好）  
7. `RDC`：推荐深度一致性（越高越好）

### 4.3.2 实验结果

根据 `data/logic_metrics_comparison.csv`，结果如下：

| 模型 | AUC | RMSE | Path(%) | PVR(%) | APC(%) | VS | RDC(%) |
|---|---:|---:|---:|---:|---:|---:|---:|
| Pure-CF | 0.5039 | 0.6520 | - | - | - | - | - |
| DKT | 0.8377 | 0.3933 | 75.22 | 24.81 | 75.19 | 0.3840 | 38.41 |
| SAKT | 0.7981 | 0.4186 | - | - | - | - | - |
| KG-SAKT | 0.8140 | 0.4112 | 68.90 | 31.30 | 68.70 | 0.4124 | 58.98 |

### 4.3.3 结果讨论

1. **预测性能维度**：  
DKT 取得最高 AUC（0.8377），说明其在“下一题是否答对”任务上拟合能力最强。KG-SAKT（0.8140）优于 SAKT（0.7981），表明图结构增强能提升注意力模型的预测表现。

2. **路径逻辑维度**：  
在 Path/PVR/APC/VS 上，DKT 本次优于 KG-SAKT；但在 RDC（深度一致性）上，KG-SAKT 明显更高（58.98 vs 38.41）。这说明 KG-SAKT 更倾向于给出与当前掌握层级匹配的“渐进式”推荐路径。

3. **多目标权衡结论**：  
本实验体现了教育推荐中的典型矛盾：  
仅优化行为预测（如 DKT）有利于 AUC；  
引入知识结构约束（KG-SAKT）有利于推荐可解释性与路径结构性。  
因此，KG-SAKT 的价值在于将“预测准确”与“教学合理”纳入统一框架，而非仅追求单一分数最优。

### 4.3.4 本章小结

本章完成了数据预处理、知识图谱构建及对比实验分析。结果表明，KG-SAKT 能够在 SAKT 基础上提升预测性能，并在推荐深度一致性方面展现结构化优势。与此同时，模型在部分先修一致性指标上仍有优化空间。下一章将对全文工作进行总结，并提出面向真实教学场景的改进方向。
